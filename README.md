# licencias-windows-generador
Generador de claves de activación de Windows


Las claves de activación se generna mediante el archivo pkeyconfig.xrm-ms que viene en cualquier instalación de Windows. Este archivo tiene información detallada de como se forma una clave valida. El presente hace muchas adecuaciones al azar (al seleccionar el grupo basado en la edición seleccionada, así como el valor de seguridad y el serial). 

Las claves llevan una firma generica, no como lo firma Microsoft, pero es una idea de como se generán .... (en algunos casos puede generarse la clave ya firmada, solo es suerte!).

Esto fue realizado con fines educativos, cualquier otro fin, es responsabilidad de quien lo utiliza.
